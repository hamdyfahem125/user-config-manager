# User Configuration Manager (CSV Version)

This is a simple User Configuration Manager built with **Python** and **Streamlit**. 
It allows users to manage settings like `theme`, `language`, `notifications` using a CSV file.

## Features

- Add new settings
- Update existing settings
- Delete settings
- View all settings
- Download settings as CSV

## Requirements

- Python 3.x
- Streamlit
- pandas

Install required packages:

```bash
pip install -r requirements.txt
